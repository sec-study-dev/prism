Mechanism summary (see metadata.json `special_mechanism_summary` for canonical version):
EigenLayer enables restaking — ETH staked on Ethereum consensus layer (or LST tokens) can be pledged to Actively Validated Services (AVSs). Operators opt into AVSs and can be slashed by the AVS slasher contract under conditions defined by each AVS; slashing events cascade to reduce podOwnerShares and weaken security guarantees across all AVSs served by the operator.

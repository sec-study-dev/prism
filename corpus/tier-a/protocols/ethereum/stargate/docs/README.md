Mechanism summary (see metadata.json `special_mechanism_summary` for canonical version):
Stargate V1 uses the Delta algorithm for unified cross-chain liquidity — balancing liquidity via credit accounting without per-pair locking. V2 adds the OFT (Omnichain Fungible Token) standard for native multi-chain token existence, AIPM routing, and Hydra batched settlement. Cross-chain credit asymmetry can be exploited by coordinating deposits and withdrawals across chains.
